HOW TO EDIT (NO CODE)
---------------------
- About: open about.html, replace 'InputAboutTextHere' text.
- Properties: open properties.html, update InputPropertyTitleHere and InputPropertyShortDescriptionHere.
  Replace /assets/property-placeholder.jpg with your real photo **using the same filename**.
- Add more properties later by copying the <article class="property-card"> ... </article> block.

DEPLOY
------
1) Upload ALL files + /assets folder to your GitHub repo (replace existing files).
2) Vercel auto-redeploys. Refresh your .vercel.app link.
3) Add smithreal.com to Vercel → Domains; set GoDaddy A + CNAME as before.
